### Structure

Generate a folder structure using the [tree](https://linux.die.net/man/1/tree) command

The command is also available on Windows, to run on GitBash you can do:

```bash
cmd //c tree
```

```text
neb-template
  |--bin
  |--doc   
  |----img 
  |--script
  |--src   
  |--test
```
